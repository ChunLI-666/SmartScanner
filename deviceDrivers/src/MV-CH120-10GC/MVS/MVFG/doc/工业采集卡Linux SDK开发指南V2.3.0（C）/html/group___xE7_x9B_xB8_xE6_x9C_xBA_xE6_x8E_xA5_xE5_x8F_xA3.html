◆ MV_FG_UpdateDeviceList()

MV_FGCTRL_API int __stdcall MV_FG_UpdateDeviceList ( IN IFHANDLE  hIface,
OUT bool8_t *  pbChanged 
)

更新指定采集卡下的相机列表

参数
hIface[IN] 采集卡句柄
pbChanged[OUT] 相机列表是否改变
返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
仅在调用该接口时内部相机列表才可更新。
