包含SDK资源初始化和释放接口

函数说明

◆ MV_FG_Initialize()

MV_FGCTRL_API int __stdcall MV_FG_Initialize ( )

初始化SDK资源

返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
在调用其他接口之前调用该接口进行初始化SDK资源。

◆ MV_FG_Finalize()

MV_FGCTRL_API int __stdcall MV_FG_Finalize ( )

结束SDK调用，销毁资源

返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
该接口调用后，SDK会释放申请的所有资源；如果要继续使用SDK接口，需要先调用 MV_FG_Initialize() 或者重新枚举采集卡。
