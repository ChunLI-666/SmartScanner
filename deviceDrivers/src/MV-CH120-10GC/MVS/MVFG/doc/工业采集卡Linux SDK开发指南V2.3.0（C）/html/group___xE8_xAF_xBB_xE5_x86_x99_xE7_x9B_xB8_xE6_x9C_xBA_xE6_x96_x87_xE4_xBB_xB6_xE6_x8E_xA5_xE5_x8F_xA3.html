◆ MV_FG_FileAccessRead()

MV_FGCTRL_API int __stdcall MV_FG_FileAccessRead ( IN PORTHANDLE  hPort,
IN OUT MV_FG_FILE_ACCESS *  pstFileAccess 
)

从相机读取文件

参数
hPort[IN] 参数句柄（可以使用采集卡/相机句柄）
pstFileAccess[IN][OUT] 文件存取结构体
返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
可能因为文件操作权限的问题导致读失败。
相机上的文件名来自FileSelector属性节点。
