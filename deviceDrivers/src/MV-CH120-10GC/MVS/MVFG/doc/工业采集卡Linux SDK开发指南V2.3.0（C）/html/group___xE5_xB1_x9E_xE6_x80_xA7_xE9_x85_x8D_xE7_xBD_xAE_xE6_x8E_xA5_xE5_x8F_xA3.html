◆ MV_FG_GetNodeAccessMode()

MV_FGCTRL_API int __stdcall MV_FG_GetNodeAccessMode ( IN PORTHANDLE  hPort,
IN const char *  strName,
OUT MV_FG_NODE_ACCESS_MODE *  penAccessMode 
)

获得节点的访问模式

参数
hPort[IN] 参数句柄（可以使用采集卡/相机句柄）
strName[IN] 节点名称
penAccessMode[OUT] 节点访问模式
返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
获取EnumEntry类型节点的访问模式，strName应传入EnumEntry_NodeName_EnumEntryName。
如获取PixelFormat枚举类型下的EnumEntry节点Mono8，strName格式为“EnumEntry_PixelFormat_Mono8”。

◆ MV_FG_SetConfigIntValue()

MV_FGCTRL_API int __stdcall MV_FG_SetConfigIntValue ( IN PORTHANDLE  hPort,
IN MV_FG_CONFIG_CMD  enConfigCmd,
IN int64_t  nValue 
)

配置自定义的整型值

参数
hPort[IN] 参数句柄（只能使用相机句柄）
enConfigCmd[IN] 配置指令
nValue[IN] 要配置的值，范围MV_FG_BAUDRATE_9600 ~ MV_FG_BAUDRATE_AUTOMAX；默认值MV_FG_BAUDRATE_115200
返回
成功，返回MV_FG_SUCCESS；失败，返回错误码
备注
仅支持Camera Link相机
