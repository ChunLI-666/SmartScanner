V2.3.0 2023/8

【功能新增】

  • 对齐Linux版本与Windows版本的功能，统一版本号为2.3.0
  • 增加CoaXPress/XoFLink/Camera Link/GigE Vision采集卡的多进程控制功能
  • 增加初始化SDK资源接口和释放SDK资源接口： MV_FG_Initialize() 、 MV_FG_Finalize()
  • 增加和完善C++示例程序

【功能优化】

  • 优化采集卡事件的性能和稳定性
  • 增加对Intel12代处理器的兼容
  • 增加对Ubuntu更高系统版本（22.04）和更高内核版本（6.1）的兼容性
  • 提升对部分第三方相机的兼容性
  • 提升对旧版本采集卡固件的兼容性
  • 修正遗留缺陷

【框架调整】

  • 优化编程引导章节，按照子功能点介绍调用流程并增加示例代码段
V1.2.0 2023/2

【功能新增】

  • 新增支持CoaXPress-6、CoaXPress-12和Camera Link接口采集卡
  • 新增注册帧缓存信息回调函数（SDK内部申请缓存方式）接口函数： MV_FG_RegisterFrameCallBackEx()
  • 新增无损解码接口函数： MV_FG_HB_Decode()
  • 新增JPEG解码接口函数： MV_FG_DecodeJpeg()
  • 新增配置自定义整型值接口函数： MV_FG_SetConfigIntValue()
  • 新增从设备读取文件接口函数： MV_FG_FileAccessRead()
  • 新增将文件写入设备接口函数： MV_FG_FileAccessWrite()
  • 新增清除GenICam节点缓存接口函数： MV_FG_InvalidateNodes()
  • 新增读取设备寄存器接口函数： MV_FG_ReadPort()
  • 新增写入设备寄存器接口函数： MV_FG_WritePort()
  • 新增注册事件回调函数接口函数： MV_FG_RegisterEventCallBack()
  • 新增注册异常信息回调函数接口函数： MV_FG_RegisterExceptionCallBack()
  • 新增示例程序：获取采集卡事件、多相机取流取图、相机重连
