错误码定义
 
正确码
名称 值 说明
MV_FG_SUCCESS 0x00000000 成功，无错误
 
通用错误码：范围0x80190001-0x801900FF
名称 值 说明
MV_FG_ERR_ERROR 0x80190001 未知的错误
MV_FG_ERR_NOT_INITIALIZED 0x80190002 模块或资源未初始化
MV_FG_ERR_NOT_IMPLEMENTED 0x80190003 功能未实现
MV_FG_ERR_RESOURCE_IN_USE 0x80190004 请求的资源正在使用中
MV_FG_ERR_ACCESS_DENIED 0x80190005 请求的操作无权限执行；例如，可能是指采集卡或相机已经被其他进程打开了
MV_FG_ERR_INVALID_HANDLE 0x80190006 无效的句柄；例如，可能是指使用了错误的句柄或者空指针句柄
MV_FG_ERR_INVALID_ID 0x80190007 无效的ID
MV_FG_ERR_NO_DATA 0x80190008 没有可以操作的数据
MV_FG_ERR_INVALID_PARAMETER 0x80190009 无效的参数
MV_FG_ERR_IO 0x80190010 IO错误
MV_FG_ERR_TIMEOUT 0x80190011 操作超时；例如，可能是指在获取图像的接口，设置的超时时间太小，导致在超时时间内未能获取到图像数据
MV_FG_ERR_ABORT 0x80190012 操作在完成之前被中止
MV_FG_ERR_INVALID_BUFFER 0x80190013 无效的缓存或者无缓存
MV_FG_ERR_NOT_AVAILABLE 0x80190014 在当前状态下，请求的资源或信息不可用
MV_FG_ERR_INVALID_ADDRESS 0x80190015 地址无效或超过范围
MV_FG_ERR_BUFFER_TOO_SMALL 0x80190016 缓存太小
MV_FG_ERR_INVALID_INDEX 0x80190017 无效的索引
MV_FG_ERR_PARSING_CHUNK_DATA 0x80190018 解析Chunk失败
MV_FG_ERR_INVALID_VALUE 0x80190019 无效的值
MV_FG_ERR_RESOURCE_EXHAUSTED 0x80190020 系统资源耗尽；例如，可能是指可用文件句柄数量达到上限
MV_FG_ERR_OUT_OF_MEMORY 0x80190021 内存申请失败；例如，可能是指系统或系统中的其他硬件（采集卡）中的内存不足
MV_FG_ERR_BUSY 0x80190022 由于请求操作的模块，此时正在执行其他操作，无法和请求的操作同时执行
MV_FG_ERR_LOADLIBRARY 0x80190023 动态库加载失败
MV_FG_ERR_CALLORDER 0x80190024 函数调用错误

名称 值 说明
MV_FG_ERR_GC_GENERIC 0x80190100 通用错误

MV_FG_ERR_GC_PROPERTY 0x80190103 属性错误
MV_FG_ERR_GC_RUNTIME 0x80190104 运行环境错误
MV_FG_ERR_GC_LOGICAL 0x80190105 逻辑错误
MV_FG_ERR_GC_ACCESS 0x80190106 权限错误
MV_FG_ERR_GC_TIMEOUT 0x80190107 超时
MV_FG_ERR_GC_DYNAMICCAST 0x80190108 转换异常
MV_FG_ERR_GC_UNKNOW 0x801901FF 未知错误
 
图像处理错误码：范围0x80190200-0x801902FF
名称 值 说明
MV_FG_ERR_IMG_HANDLE 0x80190200 图像处理库句柄错误
MV_FG_ERR_IMG_SUPPORT 0x80190201 图像处理库不支持
MV_FG_ERR_IMG_PARAMETER 0x80190202 图像处理库参数错误
MV_FG_ERR_IMG_OVERFLOW 0x80190203 图像处理库内存溢出
MV_FG_ERR_IMG_INITIALIZED 0x80190204 图像处理库操作未初始化
MV_FG_ERR_IMG_RESOURCE 0x80190205 图像处理库资源申请释放失败
MV_FG_ERR_IMG_ENCRYPT 0x80190206 图像加密错误
MV_FG_ERR_IMG_FORMAT 0x80190207 图像格式不正确或者不支持
MV_FG_ERR_IMG_SIZE 0x80190208 图像宽高不正确或者超出范围
MV_FG_ERR_IMG_STEP 0x80190209 图像宽高与step参数不匹配
MV_FG_ERR_IMG_DATA_NULL 0x80190210 图像数据存储地址为空（某个分量）
MV_FG_ERR_IMG_ABILITY_ARG 0x80190211 图像算法ABILITY存在无效参数
MV_FG_ERR_IMG_UNKNOW 0x801902FF 图像处理未知错误
