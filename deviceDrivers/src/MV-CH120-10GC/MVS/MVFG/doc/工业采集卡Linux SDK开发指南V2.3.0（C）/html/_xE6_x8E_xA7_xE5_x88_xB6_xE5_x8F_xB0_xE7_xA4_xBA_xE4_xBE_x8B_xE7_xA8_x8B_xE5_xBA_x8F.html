示例程序
 
这里列出了所有控制台示例程序：
文件名称 示例 说明
GrabImage_Callback回调取图 该示例程序展示如何使用SDK设置内部缓存并使用 MV_FG_RegisterFrameCallBack() 回调接口采集图像。在回调函数中不建议进行耗时操作，必现出现丢帧的情况。
GrabImage主动取图 该示例程序展示如何使用SDK设置内部缓存并使用 MV_FG_GetFrameBuffer() 接口采集图像。示例中展示了如何使用 MV_FG_SaveBitmap() 保存Bitmap图像。
GrabImage_UserBuffer用户注册缓存取图 该示例程序展示如何注册外部缓存，并使用外部缓存，通过 MV_FG_GetImageBuffer() 采集图像。使用外部缓存的方式，可以减少数据拷贝次数。
Event获取采集卡事件 该示例程序展示如何使用 MV_FG_RegisterEventCallBack() 接口获取采集卡接收到图像的（ReceiveImageFrameStart0）事件信息。
ChunkData获取Chunk数据 该示例程序展示如何通过 MV_FG_GetBufferChunkData() 获取块数据，并展示如何对块数据中的曝光和时间戳解析。
ConvertPixelType像素格式转换 该示例程序展示如何使用 MV_FG_ConvertPixelType() 接口进行图像格式转化。示例中统一将Mono格式的图像转为Mono8格式，彩色格式的图像转为RGB8格式。
HBDecode无损解码 该示例程序展示如何使用 MV_FG_HB_Decode() 接口进行图像无损解码。
GrabMultipleCamera_Callback多相机取图 该示例程序展示一张采集卡上多个相机如何同时采集图像。
Reconnect相机重连 该示例程序展示相机采集图像过程中断开连接后，如何恢复采集图像。
FeatureSave保存相机属性 该示例程序说明如何实现参数导入导出功能。
