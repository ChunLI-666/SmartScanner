初始化运行环境

  • 在使用工业采集卡SDK前，推荐调用 MV_FG_Initialize() 接口初始化该SDK的运行环境，以便于提前申请和分配所必须的系统资源，提升后续接口调用的流畅性。
  • 在程序退出时，可以调用 MV_FG_Finalize() 接口释放被该SDK所占用的系统资源。应当注意的是，调用该接口后，除了初始化和采集卡枚举接口外，其它所有接口都将无法被正确调用。
    下面的代码片段演示了初始化运行环境的过程：
    /*初始化运行环境*/
    MV_FG_Initialize();
    // TODO：进行设备控制，图像采集等操作
    /*在程序退出时清理运行环境*/
    MV_FG_Finalize();
