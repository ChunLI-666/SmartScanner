Detailed Description

Enumerate frame grabbers, get frame grabber information, open and close frame grabbers.

Function Documentation

◆ MV_FG_UpdateInterfaceList()

MV_FGCTRL_API int __stdcall MV_FG_UpdateInterfaceList ( IN unsigned int  nTLayerType,
OUT bool8_t *  pbChanged 
)

Update frame grabber list.

Parameters
nTLayerType[IN] Frame grabber type.
pbChanged[OUT] Whether the frame grabber list has changed.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
This API must be called before the operation of the frame grabber. The internal frame grabber list will be updated only when this API is called.
No. Frame Grabber Type Value Description
1 MV_FG_GEV_INTERFACE 0x00000001 GigE Vision
2 MV_FG_CAMERALINK_INTERFACE 0x00000004 Camera Link
3 MV_FG_CXP_INTERFACE 0x00000008 CoaXPress
4 MV_FG_XoF_INTERFACE 0x00000010 XoFLink

◆ MV_FG_ReleaseTLayerResource()

MV_FGCTRL_API int __stdcall MV_FG_ReleaseTLayerResource ( IN unsigned int  nTLayerType)

Release frame grabber resources of a specified type.

Parameters
nTLayerType[IN] Frame grabber type.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
All frame grabbers of the specified type should be closed before calling this API. Before releasing the library libMVFGControl.so via function FreeLibrary, you need to call this API to release all frame grabber resources.

◆ MV_FG_OpenInterfaceEx()

MV_FGCTRL_API int __stdcall MV_FG_OpenInterfaceEx ( IN unsigned int  nIndex,
IN int  nAccess,
OUT IFHANDLE *  phIface 
)

Open the frame grabber and specify its access mode.

Parameters
nIndex[IN] Frame grabber index, which starts from 0.
nAccess[IN] Access mode.
phIface[OUT] Frame grabber handle.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
nAccess access mode:
No. Access Type Value Description
1 MV_FG_ACCESS_READONLY 0x1 Read-only permission, which can only get node value.
2 MV_FG_ACCESS_CONTROL 0x2 Control permission. CoaXPress and XoFLink frame grabbers can only be opened with this access mode.

◆ MV_FG_OpenInterfaceWithConfig()

MV_FGCTRL_API int __stdcall MV_FG_OpenInterfaceWithConfig ( IN unsigned int  nIndex,
IN char *  pcConfigFile,
OUT IFHANDLE *  phIface 
)

Open the frame grabber with configuration file (which is saved from MV_FG_FeatureSave() ).

Parameters
nIndex[IN] Frame grabber index, which starts from 0.
pcConfigFile[IN] Configuration file path.
phIface[OUT] Frame grabber handle.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
This API needs a configuration file.
The frame grabber will be opened with control permission.
When this API is called in multiple processes to open the same frame grabber, only the first process will succeed, while other processes will report error.
