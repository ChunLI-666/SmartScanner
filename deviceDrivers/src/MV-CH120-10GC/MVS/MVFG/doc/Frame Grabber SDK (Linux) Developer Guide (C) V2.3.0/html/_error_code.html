Error Code
 
Correct Status Code
Status Code Value Description
MV_FG_SUCCESS 0x00000000 Succeeded.
 
General Error Codes (from 0x80190001 to 0x801900FF)
Error Name Error Code Description
MV_FG_ERR_ERROR 0x80190001 Unknown error.
MV_FG_ERR_NOT_INITIALIZED 0x80190002 Module or resource not initialized.
MV_FG_ERR_NOT_IMPLEMENTED 0x80190003 Function not implemented.
MV_FG_ERR_RESOURCE_IN_USE 0x80190004 The requested resource is in use.
MV_FG_ERR_ACCESS_DENIED 0x80190005 No access permission. For example, the frame grabber or the camera has been occupied by another program.
MV_FG_ERR_INVALID_HANDLE 0x80190006 Invalid handle. For example, the handle is incorrect or empty.
MV_FG_ERR_INVALID_ID 0x80190007 Invalid ID.
MV_FG_ERR_NO_DATA 0x80190008 No data.
MV_FG_ERR_INVALID_PARAMETER 0x80190009 Invalid parameter.
MV_FG_ERR_IO 0x80190010 IO error.
MV_FG_ERR_TIMEOUT 0x80190011 Timed out. For example, for image acquisition API, the timeout period might be too short to get image data.
MV_FG_ERR_ABORT 0x80190012 Operation interrupted.
MV_FG_ERR_INVALID_BUFFER 0x80190013 Invalid buffer.
MV_FG_ERR_NOT_AVAILABLE 0x80190014 The requested resource or information is unavailable.
MV_FG_ERR_INVALID_ADDRESS 0x80190015 Invalid address.
MV_FG_ERR_BUFFER_TOO_SMALL 0x80190016 Buffer is too small.
MV_FG_ERR_INVALID_INDEX 0x80190017 Invalid index.
MV_FG_ERR_PARSING_CHUNK_DATA 0x80190018 Failed to parse chunk.
MV_FG_ERR_INVALID_VALUE 0x80190019 Invalid value.
MV_FG_ERR_RESOURCE_EXHAUSTED 0x80190020 Resource exhausted. For example, the available handles reached the upper limit.
MV_FG_ERR_OUT_OF_MEMORY 0x80190021 Failed to request memory. For example, the operating system or other hardware (frame grabber) in the system is out of memory.
MV_FG_ERR_BUSY 0x80190022 Busy.
MV_FG_ERR_LOADLIBRARY 0x80190023 Failed to load DLL (dynamic link library).
MV_FG_ERR_CALLORDER 0x80190024 Incorrect order of API calls.

GenTL Related Error Codes (from -1001 to -10000)
Error Name Error Code Description
GC_ERR_ERROR -1001 Unknown error.
GC_ERR_NOT_INITIALIZED -1002 Module or resource not initialized. For example, GCInitLib is not called.
GC_ERR_NOT_IMPLEMENTED -1003 Function not implemented. For example, the function of operating the input queue and the output queue simultaneously when streaming has not been implemented.
GC_ERR_RESOURCE_IN_USE -1004 The requested resource was in use.
GC_ERR_ACCESS_DENIED -1005 No access permission. For example, the frame grabber or the camera has been occupied by another program.
GC_ERR_INVALID_HANDLE -1006 Invalid handle. For example, the handle is incorrect or empty.
GC_ERR_INVALID_ID -1007 Invalid ID.
GC_ERR_NO_DATA -1008 No data.
GC_ERR_INVALID_PARAMETER -1009 Invalid parameter.
GC_ERR_IO -1010 IO error.
GC_ERR_TIMEOUT -1011 Timed out. For example, for image acquisition API, the timeout period might be too short to get image data.
GC_ERR_ABORT -1012 Operation interrupted. For example, the waiting operation of EventGetData has been interrupted by EventKill.
GC_ERR_INVALID_BUFFER -1013 Invalid buffer.
GC_ERR_NOT_AVAILABLE -1014 The requested resource or information is unavailable.
GC_ERR_INVALID_ADDRESS -1015 Invalid address.
GC_ERR_BUFFER_TOO_SMALL -1016 Buffer is too small.
GC_ERR_INVALID_INDEX -1017 Invalid index.
GC_ERR_PARSING_CHUNK_DATA -1018 Failed to parse chunk.
GC_ERR_INVALID_VALUE -1019 Invalid value.
GC_ERR_RESOURCE_EXHAUSTED -1020 Resource exhausted. For example, the available handles reached the upper limit.
GC_ERR_OUT_OF_MEMORY -1021 Failed to request memory. For example, the operating system or other hardware (frame grabber) in the system is out of memory.
GC_ERR_BUSY -1022 Busy.
GC_ERR_CUSTOM_ID -10000 Running error.
