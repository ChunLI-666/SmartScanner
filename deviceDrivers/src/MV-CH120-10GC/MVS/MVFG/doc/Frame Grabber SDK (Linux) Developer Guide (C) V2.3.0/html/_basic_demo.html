BasicDemo

BasicDemo is a basic sample program, including examples of the calling flow of some commonly-used APIs. Sample file path: $(MVCAM_COMMON_RUNENV)\MVFG\Samples\VC\VS

BasicDemo includes:
  1. Frame Grabber: Enumerate, open and close the frame grabber.
  2. Camera: Enumerate, open and close the camera.
  3. Acquire Images: Get and set camera trigger mode, start and stop acquisition, software trigger camera.
  4. Display Images
  5. Save Images: Save images in BMP, JPG, TIFF, or PNG format.
