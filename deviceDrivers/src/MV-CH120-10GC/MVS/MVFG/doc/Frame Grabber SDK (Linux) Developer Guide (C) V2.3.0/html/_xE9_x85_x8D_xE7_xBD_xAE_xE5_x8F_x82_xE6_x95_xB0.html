Configure Features

In the SDK, general APIs can be used for getting/setting features of frame grabbers and cameras.
  • Call MV_FG_GetXMLFile() to export the XML file of the frame grabber/camera. This file contains features of the frame grabber/camera, such as description, access mode, node type, value range, and access address.
    /*Get XML file of the frame grabber/camera*/
    unsigned int nXMLBufSize = 3 * 1024 * 1024;
    unsigned char* pXMLBuf = (unsigned char*)malloc(nXMLBufSize);
    unsigned int nXMLFileLen = 0;
    nRet = MV_FG_GetXMLFile(hInterface/*hDevice*/, pXMLBuf, nXMLBufSize, &nXMLFileLen);
    Check(nRet);
    /*Write XML data to the file*/
    FILE* fp = fopen("./Interface.xml", "wb+");
    if (fp)
    {
    fwrite(pXMLBuf, nXMLFileLen, 1, fp);
    fclose(fp);
    }
  • Before calling APIs for getting and setting a property, call MV_FG_GetNodeAccessMode() and input the handle and strName of the frame grabber/camera to check if there is access permission to this property. When there is no access permission, the APIs for getting and setting the property will fail and return errors. Refer to MV_FG_NODE_ACCESS_MODE for relevant access permissions.
    /*Get access permission of a property*/
    MV_FG_NODE_ACCESS_MODE enNodeAccessMode = ACCESS_MODE_NI;
    nRet = MV_FG_GetNodeAccessMode(hInterface/*hDevice*/, "Width", &enNodeAccessMode);
    Check(nRet);
  • Before calling APIs for getting and setting a property, call MV_FG_GetNodeInterfaceType() and input the handle and strName of the frame grabber/camera to get the type of the property. Different types have different APIs for getting and setting the property. If an inappropriate API is called, an error will be returned. Refer to MV_FG_NODE_INTERFACE_TYPE for related property types.
    /*Get property type*/
    MV_FG_NODE_INTERFACE_TYPE enNodeInterfaceType = INTERFACE_TYPE_Value;
    nRet = MV_FG_GetNodeInterfaceType(hInterface/*hDevice*/, "Width", &enNodeInterfaceType);
    Check(nRet);

  • Call APIs named as MV_FG_GetXXXValue() to get property values. Call APIs named as MV_FG_SetXXXValue() to set the property values, which requires the input of the frame grabber/camera handle and the strName of the property. XXX refers to the property type, including 6 types: integer, enumeration, float, boolean, string, and command.
    The following sample codes show the process of getting/setting integer property type value of a frame grabber/camera.

    /*Get integer type value, including current value, max. and min. value, and increment*/
    MV_FG_INTVALUE stIntValue = {0};
    nRet = MV_FG_GetIntValue(hInterface/*hDevice*/, "Width", &stIntValue);
    Check(nRet);
    printf("Get Width Value, CurValue=%I64d, Max=%I64d, Min=%I64d, Inc=%I64d\n", stIntValue.nCurValue, stIntValue.nMax, stIntValue.nMin, stIntValue.nInc);
    /*Set the integer type value, with the width ranging between the max. and min. value*/
    int64_t nVal = stIntValue.nCurValue;
    int64_t nMax = stIntValue.nMax;
    int64_t nMin = stIntValue.nMin;
    int64_t nInc = stIntValue.nInc;
    nRet = MV_FG_SetIntValue(hInterface/*hDevice*/, "Width", 2048);
    Check(nRet);

    The following sample codes show the process of getting/setting enumeration property type value of a frame grabber/camera.

    /*Get enumeration type value, including current value, current symbol name, supported number of enumerations, supported values of enumeration types, and supported symbol names*/
    MV_FG_ENUMVALUE stEnumValue = {0};
    nRet = MV_FG_GetEnumValue(hInterface/*hDevice*/, "PixelFormat", &stEnumValue);
    Check(nRet);
    printf("Get PixelFormat Value, CurValue=%d, CurSymbolic=%s\n", stEnumValue.nCurValue, stEnumValue.strCurSymbolic);
    for(unsigned int i = 0; i < stEnumValue.nSupportedNum; i++)
    {
    printf("Supported Value[i=%d], Value=%d, Symbolic=%s\n", i, stEnumValue.nSupportValue[i], stEnumValue.strSymbolic[i]);
    }
    /*Set enumeration type value by entering value*/
    nRet = MV_FG_SetEnumValue(hInterface/*hDevice*/, "PixelFormat", stEnumValue.nSupportValue[0]);
    Check(nRet);
    /*Set enumeration type value by entering symbol name*/
    nRet = MV_FG_SetEnumValueByString(hInterface/*hDevice*/, "PixelFormat", stEnumValue.strSymbolic[0]);
    Check(nRet);

    The following sample codes show the process of getting/setting float property type value of a frame grabber/camera.

    /*Get float type value, including current value, max. and min. value. This type does not have an increment.*/
    MV_FG_FLOATVALUE stFloatValue = {0};
    nRet = MV_FG_GetFloatValue(hInterface/*hDevice*/, "ExposureTime", &stFloatValue);
    Check(nRet);
    printf("Get ExposureTime Value, CurValue=%4.2f, Max=%4.2f, Min=%4.2f\n", stFloatValue.fCurValue, stFloatValue.fMax, stFloatValue.fMin);
    /*Set the float type value, with the exposure ranging between the max. and min. value*/
    float fValue = 0.5 * (stFloatValue.fMax + stFloatValue.fMin);
    nRet = MV_FG_SetFloatValue(hInterface/*hDevice*/, "ExposureTime", fValue);
    Check(nRet);

    The following sample codes show the process of getting/setting boolean property type value of a frame grabber/camera.

    /*Get boolean type value*/
    bool8_t bValue = {0};
    nRet = MV_FG_GetBoolValue(hInterface/*hDevice*/, "GammaEnable", &bValue);
    Check(nRet);
    printf("The GammaEnable feature is %s\n", bValue ? "On" : "Off");
    /*Set boolean type value*/
    bValue = !bValue;
    nRet = MV_FG_SetBoolValue(hInterface/*hDevice*/, "GammaEnable", bValue);
    Check(nRet);

    The following sample codes show the process of getting/setting string property type value of a frame grabber/camera.

    /*Get string type value*/
    MV_FG_STRINGVALUE stStringValue = {0};
    nRet = MV_FG_GetStringValue(hInterface/*hDevice*/, "DeviceUserID", &stStringValue);
    Check(nRet);
    printf("Get DeviceUserID Value, CurValue=%s\n", stStringValue.strCurValue);
    /*Set string type value*/
    nRet = MV_FG_SetFloatValue(hInterface/*hDevice*/, "DeviceUserID", "Interface Name");
    Check(nRet);

    The following sample codes show the process of getting/setting command property type value of a frame grabber/camera.

    /*Set command type value*/
    nRet = MV_FG_SetCommandValue(hInterface/*hDevice*/, "UserSetLoad");
    Check(nRet);
  • Call MV_FG_FeatureSave() and MV_FG_FeatureLoad(), and input the property file name with the. hcf extension to export and import specific properties of the frame grabber/camera. Note that the properties that can be exported and imported need to have the Streamable feature or an associated feature, which is defined in the XML file of the frame grabber/camera.
    /*Export properties of a frame grabber/camera*/
    nRet = MV_FG_FeatureSave(hInterface/*hDevice*/, "./Interface.hcf");
    Check(nRet);
    /*Import properties of a frame grabber/camera*/
    nRet = MV_FG_FeatureLoad(hInterface/*hDevice*/, "./Interface.hcf");
    Check(nRet);
  • Call MV_FG_FileAccessRead() and MV_FG_FileAccessWrite(), and input user file name and device file name to read and write files from and to the frame grabber/camera. Note that the device file name comes from the FileSelector node on the frame grabber/camera, and the string value under this node can used as the input device file name.
    MV_FG_FILE_ACCESS stFileAccess = {0};
    stFileAccess.pUserFileName = "UserSet1.bin";
    stFileAccess.pDevFileName = "UserSet1";
    /*Read files from the frame grabber/camera*/
    nRet = MV_FG_FileAccessRead(hInterface/*hDevice*/, &stFileAccess);
    Check(nRet);
    /*Write files to the frame grabber/camera*/
    nRet = MV_FG_FileAccessWrite(hInterface/*hDevice*/, &stFileAccess);
    Check(nRet);
  • Call MV_FG_ReadPort() and MV_FG_WritePort(), and input register value and register address to read and write registers of the frame grabber/camera. Note that the register address can be obtained from the XML file of the frame grabber/camera. It is a node value in the format of XXX_RegAddr.
    int64_t nOutLength = 0;
    uint32_t nWidth = 0;
    /*Read the register of the frame grabber/camera*/
    nRet = MV_FG_ReadPort(hInterface/*hDevice*/, &nWidth, 0x30360, sizeof(uint32_t), &nOutLength);
    Check(nRet);
    /*Write the register of the frame grabber/camera*/
    nRet = MV_FG_WritePort(hInterface/*hDevice*/, &nWidth, 0x30360, sizeof(uint32_t), &nOutLength);
    Check(nRet);
  • Call MV_FG_SetConfigIntValue() and input configuration instructions and desired values to configure SDK custom integer parameters. For specific configuration instructions, refer to MV_FG_CONFIG_CMD. Currently, only the baud rate of CameraLink cameras can be configured, which might be expanded in later versions.
    /*Set the baud rate of a CameraLink camera to 115200 baud*/
    nRet = MV_FG_SetConfigIntValue(hDevice, CONFIG_CMD_INT64_BAUDRATE, MV_FG_BAUDRATE_115200);
    Check(nRet);
