MultipleCamera

MultipleCamera sample program shows how to control multiple frame grabbers and cameras, and how to display images from multiple cameras. Sample file path: $(MVCAM_COMMON_RUNENV)\MVFG\Samples\VC\VS

MultipleCamera includes:
  1. Initialize Frame Grabber: Enumerate, open and close the frame grabber.
  2. Initialize Camera: Enumerate, open and close the camera.
  3. Acquire Images: Start and stop acquisition.
  4. Display Images from Multiple Cameras
