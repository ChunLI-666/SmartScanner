Detailed Description

Enumerate cameras, get camera information, open and close cameras.

Function Documentation

◆ MV_FG_UpdateDeviceList()

MV_FGCTRL_API int __stdcall MV_FG_UpdateDeviceList ( IN IFHANDLE  hIface,
OUT bool8_t *  pbChanged 
)

Update the camera list of a specified frame grabber.

Parameters
hIface[IN] Frame grabber handle.
pbChanged[OUT] Whether the camera list has changed.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
The internal camera list will be updated only when this API is called.
