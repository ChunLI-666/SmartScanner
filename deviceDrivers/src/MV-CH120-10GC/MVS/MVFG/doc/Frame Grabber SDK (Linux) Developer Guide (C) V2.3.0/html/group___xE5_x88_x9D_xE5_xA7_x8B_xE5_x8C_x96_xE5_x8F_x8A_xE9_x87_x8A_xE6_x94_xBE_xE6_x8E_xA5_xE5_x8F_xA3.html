Function Documentation

◆ MV_FG_Initialize()

MV_FGCTRL_API int __stdcall MV_FG_Initialize ( )

Initialize SDK resources.

Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
This API should be called to initialize the resources before calling other APIs.

◆ MV_FG_Finalize()

MV_FGCTRL_API int __stdcall MV_FG_Finalize ( )

End the SDK calling process and revoke resources.

Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
After this API is called, the SDK will release the resources. If you still want to call the APIs, you need to call MV_FG_Initialize() or enumerate the frame grabbers again.
