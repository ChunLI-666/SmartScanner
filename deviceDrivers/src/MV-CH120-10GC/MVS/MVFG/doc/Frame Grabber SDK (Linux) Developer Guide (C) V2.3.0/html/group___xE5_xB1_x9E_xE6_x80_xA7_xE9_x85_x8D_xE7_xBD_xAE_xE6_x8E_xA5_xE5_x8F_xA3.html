◆ MV_FG_GetNodeAccessMode()

MV_FGCTRL_API int __stdcall MV_FG_GetNodeAccessMode ( IN PORTHANDLE  hPort,
IN const char *  strName,
OUT MV_FG_NODE_ACCESS_MODE *  penAccessMode 
)

Get the access mode of a node.

Parameters
hPort[IN] Handle of the object to which the parameter belongs; the object can be either a frame grabber or a camera.
strName[IN] Node name.
penAccessMode[OUT] Node access mode.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
To get the access mode of the EnumEntry type node, the input value of strName should be in the format "EnumEntry_NodeName_EnumEntryName".
For example, to get the access mode of the node MV_FG_PIXEL_TYPE_Mono8 of the enumeration MV_FG_PIXEL_TYPE, the strName should be "EnumEntry_PixelFormat_Mono8".

MV_FGCTRL_API int __stdcall MV_FG_SetConfigIntValue ( IN PORTHANDLE  hPort,
IN MV_FG_CONFIG_CMD  enConfigCmd,
IN int64_t  nValue 
)

Configure custom integer type value.

Parameters
hPort[IN] Handle of the object to which the parameter belongs; the object can only be a camera.
enConfigCmd[IN] Configuration command.
nValue[IN] Values to be set, range: MV_FG_BAUDRATE_9600 to MV_FG_BAUDRATE_AUTOMAX. Default: MV_FG_BAUDRATE_115200.
Returns
Return MV_FG_SUCCESS on success, and return Error Code on failure.
Remarks
Only Camera Link cameras are supported.
