MV_CAMCTRL_API unsigned int __stdcall MV_CC_GetSDKVersion ( )

Get the SDK version No.

Returns
Always return a version number of 4 bytes.
Main Sub Revision Test
8bits 8bits 8bits 8bits
Remarks
For example, if the return value is 0x01000001, the SDK version is V1.0.0.1.
After importing the header files and lib files, and the SDK environment is established, this API can be called directly.
