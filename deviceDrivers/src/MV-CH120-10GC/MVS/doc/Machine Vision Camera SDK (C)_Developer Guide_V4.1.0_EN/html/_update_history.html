Update History
 
Version Description
V4.1.0 Linux_March 2023
All APIs supported being used in multiple systems (Windows32/64, Linux32/64, armhf, aarch64, arm-none). The SDK function modules became consistent in Linux and Windows.
Supported controlling camera demo (BasicDemoByGenTL) and line camera demo (BasicDemoLineScan) by loading CTI files.
Added lossless decoding sample program: HighBandwithDecode.cpp
Added API for saving images: MV_CC_SaveImageEx3() , and obsoleted the old API: MV_CC_SaveImageEx2() .
Added API for saving images to files: MV_CC_SaveImageToFileEx(), and obsoleted the old API: MV_CC_SaveImageToFile() .
Added API for converting pixel format: MV_CC_ConvertPixelTypeEx(), and obsoleted the old API: MV_CC_ConvertPixelType() .
Added structure of image saving parameters: MV_SAVE_IMAGE_PARAM_EX3 .
Added structure of saving images to files parameters: MV_SAVE_IMAGE_TO_FILE_PARAM_EX .
Added structure of pixel format conversion parameters: MV_CC_PIXEL_CONVERT_PARAM_EX .
Added driver installation steps in Integrated Development Environment .
Added some error codes.
V3.2.0 Linux_July 2021
The new version uses static linking library of GenICam. And the GenICam dynamic library is not provided.
The GigE driver supports the ARM structure and two types of ARM boards: NVIDIA TX2 and ODROID-XU4.
Provided the SDK installation package (Runtime package), which includes the script of driver installation and the script of enabling read and write caching for USB3 vision cameras. The provided installation packages vary with the system (ARM or x86).
Updated the driver name to gevfilter.
Added one API for clearing the streaming data buffer: MV_CC_ClearImageBuffer()
Added one API for rotating images: MV_CC_RotateImage()
Added one API for flipping images: MV_CC_FlipImage()
Added one API for setting gamma parameters of Bayer pattern: MV_CC_SetBayerGammaParam()
Added one API for decoding the lossless compression stream into raw data: MV_CC_HB_Decode()
Added one API for setting the packet size of USB3 vision device: MV_USB_SetTransferSize()
Added one API for getting the packet size of USB3 vision device: MV_USB_GetTransferSize()
Added one API for setting the number of transmission channels for USB3 vision device: MV_USB_SetTransferWays()
Added one API for getting the number of transmission channels for USB3 vision device: MV_USB_GetTransferWays()
Added one API for getting the GVCP command timeout: MV_GIGE_GetGvcpTimeout()
Added one API for setting GVCP command retransmission times: MV_GIGE_SetRetryGvcpTimes()
Added one API for getting GVCP command retransmission times: MV_GIGE_GetRetryGvcpTimes()

V3.1.0 Linux_March 2020
