 Obsolete APIs
 APIs for getting and setting camera parameters, which are being obsoleted gradually. It is recommended to use General Configuration APIs instead. Obsoleting reasons: the newly-developed general APIs can now set specific nodes and get parameter capabilities. Those APIs with lower efficiency are replaced by new APIs. In addition, some APIs that are not commonly used in the developing process will also become obsolete gradually.
