int nStatus

[OUT] Status code returned by the device.

0x0000: Success.
0x8001: Command is not supported by the device.
0x8013: The device is not synchronized to a master clock to be used as time reference.
0x8015: A device queue or packet data has overflowed.
0x8016: The requested scheduled action command was requested at a time that is already past.
