Detailed Description

APIs for getting and setting camera parameters, which are being obsoleted gradually. It is recommended to use General Configuration APIs instead. Obsoleting reasons: the newly-developed general APIs can now set specific nodes and get parameter capabilities. Those APIs with lower efficiency are replaced by new APIs. In addition, some APIs that are not commonly used in the developing process will also become obsolete gradually.

Get image width.

Parameters
handle[IN] Camera handle..
pstValue[IN][OUT] Information structure pointer of camera width. Meaning of returned pstValue structure:
Variable Description
- unsigned int    nCurValue The current width value of the camera.
- unsigned int    nMax The maximum width value allowed by the camera.
- unsigned int    nMin The minimum width value allowed by the camera.
- unsigned int    nInc The width increment set by the camera must be a multiple of nInc, otherwise it is invalid.
Returns
Return MV_OK on success (and the structure of corresponding parameters), and return Error Code on failure.
Remarks
Other integer structure parameters API can refer to this API.
