Machine Vision Camera SDK Developer Guide Linux (C) V4.1.0

Overview

This manual mainly introduces Machine Vision Camera SDK (MvCameraSDK), which provides multiple APIs to implement the functions of image acquisition, parameter configuration, image processing, and so on.
The MvCameraSDK is compatible with GigE Vision and USB3 Vision protocols, and it can be transplanted from one camera to another without editing too much codes.
The main function of the MvCameraSDK is to control cameras, and it provides APIs, sample programs and a SDK driver.

Supported Operating System

        Linux 32-bit/64-bit:
  • Ubuntu 14.04 (32/64), Ubuntu 16.04 (32/64), Ubuntu 18.04 (32/64), Ubuntu 20.04 (32/64), Redhat7 (64), Centos7 (32/64), gcc/g++ version should be 4.4.7 or above.
  • Supported board card types for Linux ARM: NVIDIA TX2 (Ubuntu16.04), RaspberryPiB3.0+ (NOOBS_2.8.2), ODROID-XU4 (Ubuntu16.04/Ubuntu18.04)
  • X86 architect supports driver for GigE cameras.
